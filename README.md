# 🤖 Virtual AI Assistant

A powerful AI-powered virtual assistant built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) and **Google's Gemini API**. This assistant is capable of handling natural language queries, providing intelligent responses, and assisting users with various tasks in real time.

---

## 🚀 Features

- 💬 Chat-based interface for natural interaction
- 🧠 AI responses powered by **Gemini Pro API**
- 🗃️ User management and authentication
- 🌐 Fully responsive UI with React
- 📊 History/logs of user interactions
- ⚙️ Easily extendable for task automation, reminders, and more

---

## 🛠️ Tech Stack

| Frontend | Backend | Database | AI Engine |
|----------|---------|----------|-----------|
| React.js | Node.js + Express.js | MongoDB | Gemini API |
